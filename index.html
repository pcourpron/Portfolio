A web and mobile app built to help trainers track their clients, help improve decision-making and results, and increase client buy-in.
Built with React, Firebase, React-Native, Node.js and Express.

Key Coding points:




  • Trainers can only see users that have granted permissions to that trainer.
  • User authentication prevets unauthorized users from accessing other user information or the app on the front end and back end.
  • Connected to a React-native that pulls information from Apple Health kit.
  • Uses equations to calculate cardio scores and muscles stress in real time.
  • Trainers can assign workout programs to their clients, save workout programs and reuse them.
